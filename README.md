# CODSOFT - Python Programming Internship Projects

This repository contains all the projects completed during the CODSOFT Python Programming Internship.

## 📌 Projects Included

### 1️⃣ To-Do List Application
A command-line based application that allows users to:
- Add tasks
- View tasks
- Delete tasks

### 2️⃣ Calculator
A simple calculator that performs:
- Addition
- Subtraction
- Multiplication
- Division

### 3️⃣ Password Generator
Generates strong and random passwords based on user-defined length.

### 4️⃣ Rock Paper Scissors Game
A fun game where:
- User plays against computer
- Scores are tracked
- Multiple rounds supported

### 5️⃣ Contact Book
A contact management system that allows:
- Adding contacts
- Viewing contacts
- Searching contacts
- Deleting contacts
